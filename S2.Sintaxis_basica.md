# Seccion 2 : Sintaxis basica de C#, variables y tipos de datos

## Que es C#?

- Lneugaje de programacion moderno
- Orientado a objetos
- Proposito general
- Parte de .net

## Comparando tecnologias .Net

- .Net Framework (2002): apps de escritorio y servicios empresariales heredados
- .NET moderno (2020): .net 5 para adelante aplicaciones web, moviles y multiplataforma
- Xamarin (Soporte hasta Mayo 2024): desarrollo movil nativo para iOS y Android
- .Net MAUI (2022) desarrollo movil nativo

## Porque aprender c# y .Net

- Se puede desarrollar todo tipo de aplicaciones:
  - videojuegos, apps de escritorio, servicios API hasta aplicaciones web como Stack Overflow.

## Como se compila la app?

1. C# code (Program.cs)
2. C# compiler
3. Microsoft Intermediate language (MIL) esto genera ->(Program.exe, Programa.dll)
4. CLR(Common languahe runtime) -> JIT(just in time) compiler
5. Native code (CPU instructions)

- kcuando tengamos un proyecto vamos a tener 2 directorios un `/obj` que contiene un archivo objeto para cada archivo
  de codigo fuente y un directorio `/bin` encontrare el ejecutable binario para la aplicacion o biblioteca de clases.

## Primer proyecto

Creamos un directorio llamado Csharp/, luego en la terminal ejecutamos lo siguiente en una terminal:
`mkdir S2/ && cd S2` y ejecutamos el comando `dotnet new console --name HelloApp`

Para ejecutarlo uso `dotnet run`
